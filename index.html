使用说明 How To Use

⚠️ 重要：首次运行设置 ⚠️ IMPORTANT: First-Time Setup

为了使应用程序正常运行，您需要手动复制配置文件：
1. 首先启动应用程序
2. 转到应用程序中的"关于"标签页
3. 点击关于页面上显示的配置路径以打开文件夹
4. 将 pkeyconfigs 文件夹复制到此位置 For the application to function correctly, you need to manually copy the configuration files:
1. First, launch the application
2. Go to the "About" tab in the application
3. Click on the configuration path shown in the About page to open the folder
4. Copy the pkeyconfigs folder into this location

配置文件路径： Configuration paths:

Windows: 
%APPDATA%\KeyTools\config\
macOS: 
~/Library/Application Support/KeyTools/config/
没有这些配置文件，应用程序将无法正常运行！ The application will NOT function correctly without these configuration files!
1

密钥验证 Key Verification

  • 在"检测"标签页输入产品密钥（每行一个） Enter product keys in the "Check" tab (one key per line)
  • 选择验证级别（0-2） Select verification level (0-2)
  • 点击"批量检测"进行验证 Click "Batch Check" to verify
2

密钥管理 Key Management

  • 在"管理"标签页查看所有已验证密钥 View all verified keys in the "Manage" tab
  • 通过类型或产品搜索和筛选密钥 Search and filter keys by type or product
  • 右键点击任意密钥以复制或更新其状态 Right-click any key to copy or update its status
3

设置 Settings

  • 切换深色和浅色主题 Toggle between dark and light themes
  • 在中文和英文之间切换语言 Switch between English and Chinese languages
